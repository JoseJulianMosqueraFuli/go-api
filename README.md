# Simple API using GO

Simple API using gin framework for create a endpoint

## Getting Started

Requires:

- DockerDesktop (activate)
- GO

1. Enter to the project directory:

```bash
cd go-api
```

2. Build the project:

```bash
docker build
```

3. Run the container:

```bash
docker up
```
